# Sistema de Boas-Vindas Automatizado

## Tecnologias

-   ESP32
-   MicroPython
-   Sensor HC-SR04
-   Display OLED SSD1306

## Funcionamento

1.  Detecta aproximação menor que 30cm.
2.  Exibe mensagem no display.
3.  Aciona LEDs RGB.

## Requisitos

-   MicroPython instalado no ESP32
-   Componentes eletrônicos compatíveis
