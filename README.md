Este projeto desenvolve um sistema automatizado de boas-vindas utilizando ESP32 e MicroPython, com sensor ultrassônico, display OLED e LEDs RGB. Ao detectar a aproximação do cliente, exibe mensagens e ativa efeitos visuais, modernizando restaurantes e comércios. Simulado na plataforma Wokwi, demonstra aplicação prática de sistemas embarcados. Desenvolvido por Nicolas de Souza Oliveira, estudante de ADS na Faculdade de Tecnologia Prof. José Camargo - Fatec Jales, 4º semestre.
